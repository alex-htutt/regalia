# Projects — Agent Rules

## Context
Personal side projects, independent of internship work. Some tie into topics from `research/`. No deadlines unless a note says otherwise.

## Behavior in this folder

**Latitude:** More experimental than internship work. Exploring ideas, prototyping, and learning are valid goals — not everything needs to be production-ready.

**Per-project context:** Each project has a `_context_<project>.md`. Read it before doing any work in that project's folder.

**Linking research:** If a project is tied to a research topic, reference it with `[[research/<topic>/<note>]]` — don't duplicate notes, link them.

**New projects:** When starting a new project, copy `templates/_context_TEMPLATE.md` → `projects/<name>/_context_<name>.md` and add a link in `_context_projects.md`. Do not skip this step.

**Don't create files** unless asked. Propose structure in-chat first.
