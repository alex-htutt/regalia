# Course calendar

- **Create the graded milestones** you have firm dates for (final, midterms, assignment/quiz due dates). For items the syllabus only pins to a week or "see Canvas", create a clearly-labeled placeholder and tell the user to confirm the exact date.
- **Set `timeZone` explicitly.** The user may study remotely in a different zone than the school, and their calendar's default zone may not match where they currently are. School-listed times are in the school's local zone unless stated otherwise. State your assumption so they can correct it (e.g. "final 10:30 AM Pacific = 1:30 PM Eastern"). Check memory for the user's current location/timezone.
- **Add reminders** (e.g. a day before + a couple hours before) and a consistent event color so the course's items group visually.
- **Report back** exactly what you added and what still needs confirming; don't silently commit uncertain times or dates.
